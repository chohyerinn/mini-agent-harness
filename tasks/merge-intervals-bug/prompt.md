# 과제: merge_intervals 정렬 누락 버그 수정

`interval_utils.py`의 `merge_intervals(intervals)`는 `[start, end]` 형태의
구간 리스트를 받아, 서로 겹치거나 맞닿아 있는(`end == 다음 start`) 구간들을
하나로 합친 뒤, **시작 값 기준으로 정렬된** 결과를 반환해야 합니다.

현재 구현은 입력이 이미 정렬되어 있다고 가정하고 있어서, 정렬되지 않은
입력이 들어오면 겹치는 구간을 합치지 못합니다. 이를 수정하세요.

테스트는 `tests/`에 있습니다. 모든 테스트를 통과시키세요.
